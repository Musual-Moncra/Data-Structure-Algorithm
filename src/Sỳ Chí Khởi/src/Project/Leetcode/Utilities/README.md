# 🛠️ Trợ thủ Đắc lực (Utilities / Core Knowledge)

Thư mục này không chứa các thuật toán giải quyết bài toán cốt lõi (như Tìm kiếm - Search hay Sắp xếp - Sort), mà nó đóng vai trò là "Ngăn chứa công cụ" (Toolbox). Trọng tâm ở đây là các mảng kiến thức lập trình cơ bản mang tính ứng dụng cao, giúp bạn linh hoạt xử lý dữ liệu trước khi nạp vào hệ thống để chạy thuật toán.

## Danh sách Bách khoa toàn thư
1. **`FileIO_Template.c`**: Thao tác Đọc và Ghi file `.txt`. Đây là cây cầu kết nối giữa chương trình C và Dữ liệu Cứng ngoài Desktop.
2. **`String_Template.c`**: Thành thạo các kỹ thuật trong `<string.h>` (cắt, ghép, sao chép, viết hoa, tìm chuỗi con). 

## Ứng phó với mọi thể loại đề thi
Bất kỳ khi nào thầy giáo môn DSA hoặc hệ thống mLearning yêu cầu: 
- *"Cho file input.txt chứa hàng ngàn số"* -> Chạy ngay vào đây copy dòng code Load-File.
- *"Tìm tên sinh viên bất kể có viết hoa viết thường"* -> Mở liền file String Template để bế hàm ép kiểu Hoa/Thường ra xài.
