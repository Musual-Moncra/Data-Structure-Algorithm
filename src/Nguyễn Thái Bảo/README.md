# Nguyễn Thái Bảo - Cấu Trúc Dữ Liệu & Giải Thuật

## Giới thiệu

Đây là repository tổng hợp toàn bộ bài tập và tài liệu học tập môn Cấu trúc dữ liệu & Giải thuật.

## Cấu trúc thư mục

```
Nguyễn Thái Bảo/
├── 01_Recursion/              # Lab 02 - Đệ quy
├── 02_Searching/              # Lab 04 - Tìm kiếm
├── 03_Sorting/                # Lab 05 - Sắp xếp
├── 04_Linked_Lists/           # Lab 07 - Danh sách liên kết
├── 05_Advanced_Topics/        # Chủ đề nâng cao
├── Documents/                  # Tài liệu học thuật
│   ├── Academic_Documentation.md
│   └── Project_Documentation.md
├── Storage/                    # Ghi chú kỹ thuật
│   └── Technical_Notes.md
├── Logs/                       # Nhật ký học tập
│   └── activity.log
├── Tasks/                      # Files bài tập gốc
└── Gemini/                     # Solutions từ Week 4 & 5
```

## Hướng dẫn nhanh

### Biên dịch và chạy

```bash
# Vào thư mục Sorting
cd 03_Sorting/
gcc baitap01_bubble_sort_gpa.c -o baitap01 && ./baitap01

# Vào thư mục Linked Lists
cd 04_Linked_Lists/
gcc singly_linked_list.c -o linked_list && ./linked_list
```

## Tài liệu

| Tài liệu | Đường dẫn |
|----------|-----------|
| Tài liệu học thuật tổng hợp | `Documents/Academic_Documentation.md` |
| Ghi chú kỹ thuật | `Storage/Technical_Notes.md` |
| Nhật ký học tập | `Logs/activity.log` |
