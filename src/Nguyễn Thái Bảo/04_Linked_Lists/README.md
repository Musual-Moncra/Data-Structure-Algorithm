# Lab 07: Danh sách liên kết (Linked Lists)

## Nội dung
- Danh sách liên kết đơn (Singly Linked List)
- Các thao tác cơ bản:
  - Khởi tạo
  - Chèn (đầu, cuối, có thứ tự, sau node)
  - Xóa (đầu, cuối, theo giá trị, toàn bộ)
  - Tìm kiếm (iterative và recursive)
  - Duyệt và hiển thị
  - Sắp xếp (Selection Sort)

## File
- `singly_linked_list.c` - Implementation đầy đủ với menu tương tác

## Biên dịch và chạy

```bash
gcc singly_linked_list.c -o linked_list && ./linked_list
```
