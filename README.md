## 🛠 Getting Started

### Prerequisites
-   A C Compiler (e.g., `gcc`, `clang`)
-   `make` (Optional)

### Compilation
To compile any lab exercise, navigate to the specific directory and use:

```bash
gcc -o output_name LabX.c -lm
```

### Execution
Run the compiled binary:

```bash
./output_name
```
